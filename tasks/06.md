Crie a rota `GET /stats?from=&to=`

## Descrição

Essa rota retorna estatísticas do usuário autenticado em relação a treinos completos e iniciados dentro de um período.

## Requisitos Técnicos

- Formato da data recebida deve ser YYYY-MM-DD (usar z.iso.date() para validação).
- Use case deve se chamar "GetStats".
- Criar a rota em src/routes/stats.ts e registrar em src/index.ts com prefix /stats.

## Autenticação

Rota protegida.

## Request

```ts
interface Body {}

interface Params {}

interface Query {
  from: string; // YYYY-MM-DD
  to: string; // YYYY-MM-DD
}
```

## Response

```ts
interface StatusCode200 {
  workoutStreak: number;
  consistencyByDay: {
    // key: "2025-01-01" — incluir apenas dias que possuem sessão
    [key: string]: {
      workoutDayCompleted: boolean;
      workoutDayStarted: boolean;
    };
  };
  completedWorkoutsCount: number;
  conclusionRate: number;
  totalTimeInSeconds: number;
}
```

## Regras de Negócio

- Buscar todas as WorkoutSession do usuário onde startedAt está dentro do range from até to.
- Agrupar as sessões por data (parte da data de startedAt).
- consistencyByDay deve conter apenas os dias que possuem pelo menos uma sessão (dias sem atividade não entram no map).
- Se um dia de treino foi completado (completedAt preenchido), ele deve ser contabilizado tanto em workoutDayCompleted quanto em workoutDayStarted.
- Dias de descanso (isRest = true) também devem ser contabilizados caso eles sejam iniciados e/ou concluídos.
- Se um dia de treino foi apenas iniciado (sem completedAt), ele deve ser contabilizado apenas em workoutDayStarted.
- conclusionRate: completedWorkoutsCount / totalDeSessões (0 se não houver sessões).
- totalTimeInSeconds: soma de (completedAt - startedAt) em segundos para todas as sessões completadas.
- workoutStreak: número de dias em sequência que o usuário completou algum dia de treino. Exemplo: Se o plano de treino tem os dias de treino segunda, terça, quarta, quinta e sexta-feira, e o usuário completou todos (mesmo que algum deles seja dia de descanso) ele terá um streak de 5.
