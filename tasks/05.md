# Template de Prompt para Criar uma Rota de API

Crie a rota `GET /workout-plans/:id/days/:id`

## Descrição

Essa rota deve retornar o dia de treino recebido, seus exercícios e, para cada exercícios, suas sessões (workout sessions).

## Requisitos Técnicos

N/A

## Autenticação

- Rota protegida.
- O usuário precisa ser dono do plano de treino recebido.
