# Template de Prompt para Criar uma Rota de API

Crie a rota `PATCH /workout-plans/{id(uuid)}/days/{id(uuid)}/sessions/:id(uuid)`

## Descrição

Ela atualiza uma sessão de treino específica.

## Autenticação

- Rota protegida.
- O usuário precisa ser dono do plano de treino recebido.
