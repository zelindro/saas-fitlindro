# Template de Prompt para Criar uma Rota de API

Crie a rota `GET /home/{date}`

## Descrição

Essa rota deve retornar os dados necessários do usuário autenticado para preencher a página inicial da aplicação.

## Requisitos Técnicos

- Receba uma data no formato YYYY-MM-DD como param na rota.
- Use a biblioteca dayjs para manipular e formatar datas.

## Autenticação

Rota protegida.

## Request

```ts
interface Body {}
```

```ts
interface Params {
  date: string;
}
```

```ts
interface Query {}
```

## Response

[Descreva o retorno esperado e o status code, por exemplo:]

```ts
interface StatusCode200 {
  activeWorkoutPlanId: string;
  todayWorkoutDay: {
    workoutPlanId: string;
    id: string;
    name: string;
    isRest: boolean;
    weekDay: string;
    estimatedDurationInSeconds: number;
    coverImageUrl?: string;
    exercisesCount: number;
  };
  workoutStreak: number;
  consistencyByDay: {
    // key: "2025-01-01" — incluir TODOS os dias, mesmo os que não possuem sessão
    [key: string]: {
      workoutDayCompleted: boolean;
      workoutDayStarted: boolean;
    };
  };
}
```

## Regras de Negócio

- A partir da data recebida, calcular o início (domingo) e fim (sábado) da semana em UTC.
- Para recuperar o treino do dia, faça o seguinte: 1) Procure pelo workout plan ativo; 2) Procure nos dias de treino do workout plan ativo o dia corresponde à data atual (recebida na rota).
- workoutStreak: número de dias em sequência que o usuário completou algum dia de treino (incluindo dia de descanso). Exemplo: se o plano de treino ativo tem segunda, terça, quarta e quinta-feira e o usuário completou todos (incluindo descanso) o streak dele é 4.

### Para montar o `consistencyByDay`

- Buscar todas as WorkoutSession do usuário onde startedAt está dentro do range domingo 00:00:00 até sábado 23:59:59 da semana correspondente.
- Agrupar as sessões por data (parte da data de startedAt).
- Se uma sessão de treino foi completado (completedAt preenchido), ela deve ser contabilizada tanto em workoutDayCompleted quanto em workoutDayStarted.
- Se uma sessão de treino foi apenas iniciado (sem completedAt), ela deve ser contabilizada apenas em workoutDayStarted.
