## Tarefa

Crie os seguintes use cases:

### UpsertUserTrainData

Esse use case deve criar ou atualizar os dados de treino de um usuário.

Parâmetros:

```ts
interface InputDto {
  userId: string;
  weightInGrams: number;
  heightInCentimeters: number;
  age: number;
  bodyFatPercentage: number; // 1 representa 100%
}
```

Resposta:

Ele deve retornar esses mesmos dados.

### GetUserTrainData

Deve retornar os seguintes dados:

```ts
interface OutputDto {
  userId: string;
  userName: string;
  weightInGrams: number;
  heightInCentimeters: number;
  age: number;
  bodyFatPercentage: number; // 1 representa 100%
}
```

Crie também uma rota `GET /me` que chama esse use case e retorna os mesmos dados.

### Regras

- A rota `GET /me` deve retornar `null` (com status 200) caso os dados de treino do usuário ainda não existam, e **não** um erro 404.
